## 📂 Profile Categories with Examples

### Minimal & Clean Style

**What it looks like:**
- Simple header with name and title
- Short About Me (5-6 lines max)
- Clean badge layout
- Stats at the bottom
- No flashy animations

**Best for:** Professionals, senior developers, open source maintainers

**Template:** [templates/developer.md](../templates/developer.md)

---

### Student / Beginner Style

**What it looks like:**
- Friendly greeting with emoji
- Learning goals prominently mentioned
- Shows projects honestly (even if small)
- Contact info and LinkedIn visible
- A visitor counter

**Best for:** Students, bootcamp graduates, career switchers

**Template:** [templates/student.md](../templates/student.md)

---

### Creative / Animated Style

**What it looks like:**
- Animated header with capsule-render
- Typing animation
- Colored streak stats
- Activity graph
- Trophy section

**Best for:** Developers who want to stand out visually

**Template:** [templates/developer.md](../templates/developer.md)
