## 📝 Customization Guide

| Placeholder | What to Put |
|-------------|------------|
| `YOUR NAME` in capsule render | URL-encoded name (spaces = `%20`) |
| `YOUR_USERNAME` | GitHub username (appears many times) |
| `[Your Role]` | e.g., Senior Backend Engineer |
| `[X] years` | Your years of experience |
| `[Company Name]` | Current employer or "freelance" |
| `YOUR_REPO_1` / `YOUR_REPO_2` | Your best repository names |
| Social links | Replace all `YOUR_*` with actual URLs |

## 💡 Pro Developer Tips

1. **Quantify achievements** — "10M+ users" beats "large scale"
2. **Show private contribution stats** — add `&count_private=true` to stats URL
3. **Add a resume link** — make it easy for recruiters
4. **Update regularly** — stale profiles hurt more than they help
5. **Remove widgets you don't qualify for** — empty trophies look bad
